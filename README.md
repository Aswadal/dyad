# Dyad Terminal

Enhanced Dyad environment with integrated terminal support.

## Features
- Full terminal emulation in browser
- Dyad Lisp editor
- Shell command execution
- Deployable to Netlify with one click

## Quick Deploy
[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/YOUR_USERNAME/dyad-terminal)

## Usage
1. Visit your Netlify URL
2. Left panel: Write Dyad Lisp code
3. Right panel: Execute terminal commands
4. Use `(shell "command")` from Dyad
